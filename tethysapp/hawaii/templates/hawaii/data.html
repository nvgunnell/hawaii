Data Used in Geoprocessing

Digital Evelation Model (DEM)

The primary data that is used in the geoprocessing of this application is DEM data of the Hawaiian Islands. DEM data is data that provides elevation data at a given resolution. The resolution of the data can range from 30-meters to 0.5-meters. DEM data is widely available and is regularly updated. These two facts made the use of DEM data the best option for calculating mass flux in the Hawaiian Islands. The image below is an map that was created to show the extents of the DEM data that will be used in this application.


User Created Data

To preform the mass flux calculation it is necessary to have a shapefile of points along the ridge of the canyon where the mass flux is being calculated. This data will be created by the user on an interactive map that will be included as part of this application. The points created by the user are used to calculate the mass flux of the area of interst. The area of interst will also be defined by the user using an interactive map. The user will be able to create a ploygon on the interactive map to define the area of interest.
